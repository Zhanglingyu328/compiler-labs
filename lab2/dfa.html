实验说明与报告亮点

本平台将第一次实验的 DFA 自动机实验与第二次实验的类 C 语言词法分析器实验融合，展示从理论自动机到编译器前端 scanner 的完整过程。

DFA 五元组 字符串判定 合法串枚举 Token 扫描 错误定位 统计分析
模块一：DFA 可视化实验
用于展示确定有限自动机的五元组、状态转移、字符串判定和合法字符串枚举。支持内置案例和自定义 DFA。
模块二：类 C 词法分析器
用于扫描类 C 语言源代码，识别关键字、标识符、整数、实数、运算符、界符，并输出 token 序列。
DFA 与 scanner 的关系
词法分析器本质上可以看作多个 DFA 的组合。每类 token 都可以由正则式描述，再进一步转换为 NFA/DFA 进行识别。
高分拓展点
本系统加入了自定义 DFA、状态转移表、token 表格展示、行列号定位、词法错误恢复、token 数量统计和 Web 可视化页面。
报告建议写法：本实验在命令行 scanner 基础上进一步实现了 Web 综合可视化平台，使 DFA 的理论实验与类 C 语言词法分析器的工程实现形成对应关系，体现了正则式、自动机和编译器前端之间的联系。
