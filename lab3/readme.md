# 编译器设计专题实验三：LR(0) 项目集规范族构造

## 一、实验目标

本实验实现给定文法的 LR(0) 项目集规范族构造。程序支持从文件读取扩展巴科斯范式形式的文法规则，自动完成文法增广、Closure 闭包计算、Goto 状态转移计算，并输出所有项目集及其内核项。

## 二、实验内容

1. 输入文法规则；
2. 自动增广文法；
3. 构造初始项目集 I0；
4. 计算 Closure；
5. 计算 Goto；
6. 生成 LR(0) 项目集规范族；
7. 输出内核项、闭包项、状态转移关系；
8. 检查 LR(0) 冲突；
9. 导出 JSON 和 DOT 文件；
10. 选做：串联实验一和实验二的词法分析结果。
